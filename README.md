# Campo-Q Lab

O **Campo-Q Lab** é o laboratório exploratório da **Teoria do Evento (TE)**.
Este repositório funciona como **índice conceitual, organizacional e epistemológico** dos experimentos públicos associados à teoria.

Aqui **não há código executável**.
Este repositório existe para **organizar, contextualizar e referenciar** os experimentos, preservando sua leitura correta e auditável.

---

## Objetivo do Campo-Q Lab

O Campo-Q Lab tem como objetivo investigar, por meio de **toy models explícitos**, se determinados fenômenos observados na cosmologia contemporânea podem ser reinterpretados como **efeitos de regimes distintos de coerência informacional**, conforme proposto pela Teoria do Evento.

Os experimentos aqui reunidos:

- não pretendem provar a Teoria do Evento;
- não substituem a física cosmológica padrão;
- não introduzem novos dados observacionais;
- não realizam ajustes paramétricos finos.

Eles servem para **tornar explícitas tensões, transições e inconsistências conceituais** já presentes nos dados públicos.

---

## Estrutura dos Experimentos

Cada experimento do Campo-Q Lab:

- é **autônomo**;
- é **executado isoladamente**;
- possui seu próprio repositório;
- contém:
  - um `README.md` explicativo,
  - um script simples (`experimento.py`),
  - um arquivo de metadados (`metadata_expXX.json`).

Não há dependência lógica ou computacional entre os experimentos.

## O que este laboratório **não** afirma

O Campo-Q Lab **não afirma**:

- que a Teoria do Evento esteja comprovada;
- que a tensão de H₀ esteja resolvida;
- que exista erro instrumental oculto específico;
- que um único regime cosmológico global seja inválido.

Ele afirma apenas que **a coexistência de regimes distintos é conceitualmente consistente** e merece investigação explícita.
